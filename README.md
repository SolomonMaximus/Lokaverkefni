---

# What this site does...

-This is a site made for browsing recipes, you can filter it by a few categories, and you may search what you like. Open up a certain recipe to get more information about the recipe

# Pages and features

Home page.

In the top left corner is a ☰ symbol that can be clicked to access recipes and go back home.

The "Bon appétit" in the top middle can be clicked to go back home.

Recipes page with a list and images of recipes.

Recipe pages with details.

Recipes can be filtered by categories in the dropdown box.

Search bar so that you can look for what you want to find, if nothing is found, a message comes up and tells you to try again.
